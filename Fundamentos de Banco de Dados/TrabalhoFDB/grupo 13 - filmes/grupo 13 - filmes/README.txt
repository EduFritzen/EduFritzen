O banco de dados armazena informações de todos os diretores a trabalharem na série Doctor Who até 2019.
Os dados armazenados para cada tabela são:
Diretor: ID e o nome.
História: ID, nome, o ID do diretor (certas histórias possuem múltiplos diretores, para manter o banco de dados simples foi selecionado apenas o primeiro), e a temporada a qual ela pertence.
Episódio: Id, nome, o ID da história a qual pertence, a data de transmissão e a duração.
dados obtidos de: https://api.catalogopolis.xyz/docs/restapi/index.html
